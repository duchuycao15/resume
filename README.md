🌐 A Resume on the Web
I built this site using plain HTML5 and CSS3 so that anyone, anywhere, can view my résumé with just a click. The page is fully responsive, meaning it looks great on laptops, tablets, and phones alike—no extra JavaScript or frameworks required! 📱💻

Link: https://duchuycao15.github.io/resume/


📄 What You’ll Find Inside
The résumé is laid out in a clean, single‑page design that highlights the key elements of my professional story:

🎓 Education & Certifications – Degrees earned, online courses completed, and industry certifications (e.g., AWS, PMP).

🛠️ Technical Skills – Programming languages, frameworks, tools, and cloud platforms I’m comfortable with.

💼 Work Experience – Roles, responsibilities, and quantifiable achievements from my past positions.

📈 Projects & Case Studies – Links to live demos, GitHub repos, or downloadable PDFs that showcase real‑world problem solving.

🤝 Soft Skills & Interests – Team collaboration, leadership moments, and hobbies that add depth to the professional portrait.

🔗 Easy Navigation & Downloads

A sticky navigation bar lets you jump straight to the section you’re interested in, and there’s a “Download PDF” button so you can keep a copy for future reference or share with recruiters. 🎁
